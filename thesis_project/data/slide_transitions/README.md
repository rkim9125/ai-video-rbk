# Slide Transition Annotations

Create one file per lecture:

- `lecture1_slides.txt`
- `lecture2_slides.txt`
- `lecture3_slides.txt`
- `lecture4_slides.txt`

Each line should contain a timestamp in `HH:MM:SS` format.
Optional comments can be added after the timestamp.

Example:

00:05:10
00:12:45 intro -> methods
00:24:30
